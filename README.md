# dbf2sql
Cette commande, écrite en Pascal (Turbo Pascal), permet de convertir un fichier .DBF (DBase III) en format SQL.

<h3>Syntaxe</h3>

<b>DBF2SQL</b> <i>source</i>.DBF

<h3>Paramètres</h3>


<table>
  <tr>
    <th>Nom</th>
    <th>Description</th>
  </tr>
  <tr>
    <td><i>source</i></td>
    <td>Ce paramètre permet d'indiquer le fichier à convertir.</td>
  </tr>
</table>
